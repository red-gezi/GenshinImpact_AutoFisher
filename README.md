# 产品介绍

# 使用说明
打开exe文件（注意，一定要右键，以管理员模式启动，不然原神无法接受模拟输入）
选择显示目标标签
点击open按钮
ps:
+ 可在采集界面检查捕获图片是否完整，上层为采集图片，下层为特征的对比结果，如果是黑屏则是没有用管理员权限打开
+ 标志点与实际位置不匹配可以通过校准页面的数值进行调整
