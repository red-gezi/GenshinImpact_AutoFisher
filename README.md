+ 通过win32api采集原神进程对应的画面，提取钓鱼槽进行分析和模仿鼠标点击

打开exe文件
进行钓鱼，按p暂停，然后根据分析结果调整坐标和参数减少干扰
点击open按钮开始自动钓鱼
